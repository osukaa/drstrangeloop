Callbacks

Es una función que se pasa como argumento para que se ejecute en algún futuro.


							const fs = require('fs');
							fs.readFile('/some/path/to/file', (err, data) => {
								if (err) throw err;
								console.log(data);
							});

Promises

Es un remplazo para un resultado que todavía no está disponible


							fs.readFileAsync('/some/path/to/file')
								.then((data) => {
									console.log(data);
								})
								.catch((err) => {
									console.error(err);
								});

async / await

El proposito de async/await es simplificar el uso de promesas.


							try {
							  const data = await fs.readFileAsync('/some/path/to/file')
							} catch (e) {
							  console.error(e);
							}

Es una combinación de las mejores ideas del patrón Observer, Iterator y programación funcional


						var Rx = require('rx');
						var walk = require('walk');

						var walkerSource = walk.walk(__dirname);
						var walker = Rx.Observable.fromCallback(walkerSource.on);
						var source = walker('file');

						console.log('walk test');
						source.subscribe(
						function(root, stats, next) {
							console.log(stats.name);
							next();
						},
						function(root, stats, next) {
							console.log(stats.name);
							next();
						},
